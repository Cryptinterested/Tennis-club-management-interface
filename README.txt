

Pour exécuter le programme :  

 1- Rendez vous ici : Tennis_POO/INTERGRAPH/bin/Debug/

 2- Double-cliquez sur l'application INTERGRAPH.exe (avec le logo balle de tennis)

 3- Vous arrivez sur l'accueil du programme (Onglet "Menu")

 4- Vous pouvez alors naviguez à travers les différentes sections via les onglets 
    proposés en haut :
  
  Onglet "Inscription" : - Dans cette section est destinée à la création de membres 
  			 - Les entraineurs sont déjà prédéfinis

  Onglet "Espace Administratif" : - Dans cette section munissez vous du mot de passe : "tennis2020", pour y accéder 
				  - Vous arrivez sur le sous-onglet "Ajout Cours au planning" celle-ci permet de créer un créneau de cours 
				  - Sous-onglet "Compétition": vous pouvez créer une compétition et choisir l'option simple ou équipe 
  				  - Sous-onglet "Tournoi des Familles" : permet de créer un événement comme un tournoi des familles ou autres 
				  - Sous-onglet "Stage" : la création de stages se fait via ce sous-onglet
				  - Sous-onglet "Membres" : Permet l'affichage des membres selon différents critères
				  - Sous-onglet "Former Equipe" : Permet de définir le capitaine d'une équipe 
				  - Sous-onglet "Statistiques" : vous pouvez étudier les performances du club et des membres
				  - Sous-onglet "Cotisation" : Permet de gérer les impayés et modifier le statut des cotisations
				  - Sous-onglet "Statistiques" : Suivant différents critères vous accédez aux performances de votre club

  Onglet "Membre" : - Cet onglet est accessible par les membres du club, munissez vous de votre numéro de licence 
		    - Sous-onglet "Statut" : Vous donne votre statut au niveau du club selon que vous soyez membre loisir ou compétition 
		    - Sous-onglet "Former Equipe" : Cette sous-section permet au capitaine (choisi par un membre administratif) de sélectionner 
                      les membres de son équipe 
		    - Sous-onglet "Reserver Cours" :  Via cette onglet il est possible de réserver un cours 

  		    


  PS : Pour tester ce programme veuillez inscrire au minimum 5 membres (indispensable si vous souhaitez créer une équipe)
